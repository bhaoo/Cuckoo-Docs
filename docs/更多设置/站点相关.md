## 底部信息

前往 **“设置外观-功能-底部信息”** 处填写即可（支持html）

## 站点版权

### 可用变量

| 变量名  | 描述 | 示例 |
|--------|-----|----|
| {site} | 站点链接 |https://cuckoo.owo.show/|
|{name}|基础设置->昵称|Bhao|
|{year}|当前年份|2023|

### 例子

&copy; 2023 <a href="https://cuckoo.owo.show/">Bhao</a>
```
&copy; {year} <a href="{site}">{name}</a>
```
