# 短代码

## 可扩展面板

### 使用方式

```
[pl title="标题" summary="副标题"]内容[/pl]
```

!> 需不显示副标题为空（summary=""）即可，其他也是如此

## Bilibili 视频解析

### 使用方式

```
[bili av="" bv=""]
```

!> AV/BV 号填写其中一个就行

### 例子

```
[bili av="" bv="BV1uc411h7iC"]
```
